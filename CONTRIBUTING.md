# Contributing to AlphaBot2_APP

Thanks for your interest in contributing!

## How to contribute

1. Fork this repository  
2. Clone your fork (`git clone https://github.com/YourUser/AlphaBot2_App.git`)  
3. Create a new branch (`git checkout -b feature/YourFeature`)  
4. Make your changes & add tests/documentation  
5. Submit a Pull Request and describe your changes

## Code style

- Android app in Kotlin (MVVM)  
- Python scripts for robot control  
- Follow naming conventions, comment your code

## Issues

If you find a bug or want a new feature, open an issue. Please provide steps to reproduce.
